# Pig Game 🎲

## Description
Pig Game is a simple browser-based dice game built using JavaScript. Players take turns rolling a dice to accumulate points, with the goal of reaching 100 points first.

## Features
- Random dice roll generation
- Current and total score tracking
- Player switching logic
- Win condition (first to 100 points)
- Game reset functionality

## Tech Stack
- HTML
- CSS
- JavaScript (ES6+)

## How to Play
- Roll the dice to add to your current score
- If you roll a 1, your current score resets and turn switches
- Hold to add current score to total score
- First player to reach 100 wins
